# Order Summary Card

A **responsive Order Summary Card** built using **HTML & CSS**.  
This project is a [Frontend Mentor](https://www.frontendmentor.io/) challenge that helps practice translating a design into a functional webpage.

---

## Description

This component simulates an order summary interface where users can:

- View order details (plan type, price)  
- Change their subscription plan  
- Proceed to payment or cancel the order  

The project focuses on **mobile-first responsive design**, semantic HTML, and **hover interactions** for a better user experience.

---

## Features

- Fully responsive card layout  
- Hover effects on interactive elements (`Change` link and buttons)  
- Clean and semantic HTML structure  
- Reusable CSS classes for maintainable styling  

---
